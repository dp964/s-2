TEX TO HTML RETRIEVER

About this app

This app was created to parse TEX files into HTML. The script works as follows:
  1. Extract .tex file from source as raw text
  2. Wrap raw text in a XHTML skeleton/container, parsing any errors still as raw text
  3. Use a XML parser to convert the XHTML into raw HTML
  4. Load MathJax script and replace $...$ pattern with regex
  5. Run regex find/replace patterns to replace non-math LATEX commands into HTML tags
  6. Post-processing and display HTML output using a predefined CSS template

Custom Configurations

The script can be configured by adding parameters to the script url as follows

?serve: the id of the tex file in Google Drive

?base: the template served by the script. Some templates may take files from another server instead of Google Drive if configured correctly.
  drive [default]: Retrieves the file from Google Drive
  drive_withedit: Retrieves the file from Google Drive with options to edit and download the TEX file.
  thsc: Retrieves the file from /s/tex/

?sourcecode: whether to display the sourcecode.
  true: Displays the sourcecode of the file
  [default]: Displays the final output

?render: if "sourcecode" is set to true, this will determine at what point of processing, the code is presented
  0: Displays the content of the retrieved TEX file
  1: Displays the content of XHTML skeleton before XML processing
  2 [default]: Displays the HTML content used for the template
